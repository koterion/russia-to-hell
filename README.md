## Mac/Ubuntu/Linux:
* install - `bash ddos-unix.sh -m install`
* restart - `bash ddos-unix.sh -m reinstall`
* stop - cntr + c and `bash ddos-unix.sh -m stop`

### Parameters:
* -m - mode
* -n - number of containers default 50 (if problem with process add -n 10)

## Windows
`ddos.bat`

## Require packages:
* docker
* docker-compose
* python
